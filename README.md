🐾 Sistema Sustentável de Identificação Animal
Projeto que propõe uma solução acessível, sustentável e eficiente para identificação de animais domésticos, utilizando coleiras com QR Code produzidas a partir de materiais reciclados.

📌 Sobre o Projeto
O sistema busca resolver problemas comuns na identificação animal no Brasil, como:

Alto custo da microchipagem
Falta de acesso para famílias de baixa renda
Grande número de animais perdidos ou abandonados
Dificuldade de identificação em situações de emergência (ex: desastres naturais)

A proposta oferece uma alternativa simples e de baixo custo, promovendo tecnologia + sustentabilidade + bem-estar animal.

🚀 Solução Proposta
🧷 Coleiras com QR Code

QR Code com informações do animal e tutor
Leitura rápida via smartphone (sem equipamentos especiais)
Redirecionamento para contato ou sistema online

♻️ Produção Sustentável

Uso de plástico reciclado (garrafas PET)
Impressão 3D (filamento PETG)
Produção local (reduz impacto ambiental)

💰 Baixo Custo

Muito mais barato que microchipagem
Viável para adoção em larga escala


🐶 Aplicações

Cães e gatos
Aves (com anilhas QR ou magnéticas)
Outros animais domésticos


🌍 Impacto Social
O projeto pode ajudar:

ONGs de proteção animal
Prefeituras
Bombeiros em resgates
Clínicas veterinárias

Além disso, contribui para:

Redução de abandono
Identificação rápida em desastres
Políticas públicas de proteção animal


🎯 Objetivo Maior
A proposta está alinhada com iniciativas de:

Sustentabilidade
Proteção animal
Inclusão social
Uso consciente de tecnologia


📣 Conclusão

“Não é apenas sobre identificar animais, mas transformar resíduos em soluções acessíveis que conectam tecnologia, sustentabilidade e cuidado com a vida.”
